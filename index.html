Webpack 4 without configuration

Using Webpack 4, you can begin to use Webpack without creating a webpack.config.js file.

Step 1

To do that, you first needed to initialize a new project.

    
    npm init -y

That will create a package.json file in our project directory with default settings.

Step 2

Now you can install webpack, webpack-cli, and webpack-dev-server.

    
    npm install webpack webpack-cli webpack-dev-server

Step 3

Next, you needed to open package.json and under scripts, create two new scripts:

    
    "scripts": {
        "build": "webpack --mode development",
        "start:dev": "webpack-dev-server"
    },

We'll get back to how to run these scripts in a minute.

Step 4

Next, create an index.html file with boilerplate html and add a script tag just before the closing body tag that links to our main JavaScript file. We'll create that file next.

    
    <!DOCTYPE html>
    <html lang="en">
    
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <meta http-equiv="X-UA-Compatible" content="ie=edge">
            <title>Webpack Test</title>
        </head>
        
        <body>

        <script src="main.js"></script>
        </body>
        
    </html>
